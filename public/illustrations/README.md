# SEAPEDIA Illustrations

Folder ini berisi ilustrasi SVG untuk halaman auth SEAPEDIA frontend.

## Required Files

| File | Halaman | Deskripsi |
|------|---------|-----------|
| `login.svg` | Login | Ilustrasi untuk halaman masuk |
| `register.svg` | Register | Ilustrasi untuk halaman pendaftaran |

## Cara Mendapatkan Ilustrasi dari undraw.co

1. Buka [undraw.co](https://undraw.co/)
2. Cari ilustrasi yang sesuai:
   - **Login**: cari "login", "welcome", "authentication"
   - **Register**: cari "signup", "register", "create account"
3. Klik ilustrasi pilihan → klik **"Download SVG"**
4. Simpan file sebagai `login.svg` dan `register.svg` di folder ini

## Catatan

- Gaya ilustrasi: flat, minimal, modern
- Background ilustrasi menggunakan `#F1EFE8` (warna brand SEAPEDIA)
- Pilih ilustrasi dengan warna yang selaras dengan brand: Navy (#042C53), Teal (#0F6E56)
- Ukuran direkomendasikan: 400-500px width, aspect ratio bebas
- Pada mobile (< 768px), kolom ilustrasi akan disembunyikan

## Troubleshooting

Jika ilustrasi tidak tampil, pastikan:
1. File benar-benar ada di folder ini
2. Nama file sesuai (case-sensitive): `login.svg`, `register.svg`
3. File SVG valid (bisa dibuka di browser langsung)
